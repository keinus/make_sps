Make SPS

사용 방법

1. 폴더 구조 생성

각 장비별로 별도의 폴더를 생성합니다. 각 폴더 내에는 CSU별로 소스코드 파일이 저장됩니다.

.
├── device-1
│   ├── test1
│   ├── test2
│   ├── test3
│   └── test4
└── project.yaml

2. project.yaml 파일 작성

각 장비별 폴더에 project.yaml 파일을 하나씩 작성합니다. 이 파일은 해당 장비의 프로젝트 정보를 포함하고 있습니다.

project:
  device: HDEV-001
  version: 1.0.0
  partnumber: Q2350911516
  checksum_type: SHA256
  csu:
    - csu: Test1 (D-AAA-SFR-001)
      dir: test1
    - csu: Test2 (D-AAA-SFR-002)
      dir: test2
    - csu: Test3 (D-AAA-SFR-003)
      dir: test3
    - csu: Test4 (D-AAA-SFR-004)
      dir: test4

3. 파일 압축

  1. 장비별 폴더를 ZIP 형식으로 압축합니다.
  2. device-1 폴더를 선택합니다.
  3. 압축 프로그램을 사용하여 해당 폴더를 ZIP 파일로 압축합니다.

4. UI에서 파일 업로드

압축한 ZIP 파일을 UI에서 업로드합니다.
